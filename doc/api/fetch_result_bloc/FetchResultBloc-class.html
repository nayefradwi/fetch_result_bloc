FetchResultBloc

FetchResultBloc<P, T> class abstract

An abstract BLoC (Business Logic Component) for fetching data.

It handles common states like

  • initial
  • loading
  • refreshing
  • loaded
  • error. Extend this class to implement specific data fetching logic by overriding getResult.

Type parameters:

  • P: The type of parameters required for fetching the data.
  • T: The type of the data to be fetched.
