fetch_result_bloc library

A Dart package providing abstract BLoC and Cubit components for simplifying common data fetching and state management patterns.

This library offers base classes (FetchResultBloc, FetchResultCubit) that handle typical states (initial, loading, refreshing, loaded, error) and events, allowing developers to focus on the specific data retrieval logic. It also includes helper classes for events (FetchResultEvent), states (FetchResultState), and error customization (FetchResultErrorOptions).
