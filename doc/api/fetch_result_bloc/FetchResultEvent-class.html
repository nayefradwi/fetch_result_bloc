FetchResultEvent

FetchResultEvent<P> class sealed

Represents events for fetching or refreshing data in a FetchResultBloc.

This is a sealed class, meaning all specific event types are defined as subclasses herein. It holds the parameters required for the fetch operation.

Type parameters:

  • P: The type of parameters required for the fetch operation.
